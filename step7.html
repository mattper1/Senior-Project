Step 7: Install GPU

Materials Needed

  • Graphics Card (GPU)
  • Phillips Head Screwdriver

Instructions

  1. Install the GPU into the PCIe Slot

    2. Align the GPU with the PCIe slot on the motherboard and gently press it down until it clicks into place.

  2. Screw the GPU into the Case

    3. Use screws to secure the GPU to the case, ensuring it is firmly in place to prevent any movement.
