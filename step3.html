Step 3: Install RAM

Materials Needed

  • RAM Sticks
  • Motherboard

Instructions

  1. Locate the RAM Slots on the Motherboard

    2. Find the RAM slots on your motherboard, usually labeled DIMM 1, 2, 3, and 4. For dual-channel performance, it's best to install your RAM in slots 1 and 3, or 2 and 4.

  2. Align the RAM Stick

    3. Take one RAM stick and align it with the slot. The RAM has a notch on the bottom that matches the slot, so it can only be inserted one way.

  3. Install the RAM Stick

    4. Carefully insert the RAM stick into the slot. Apply firm, even pressure until you hear a click, indicating that the RAM is fully seated and the side clips have locked it in place.

Important Tips

  • Slot Order: If you’re using two RAM sticks, use slots 1 and 3 or 2 and 4 to enable dual-channel mode for better performance.
  • Listen for the Click: Push the RAM down firmly until you hear a click, signaling it is securely installed.
