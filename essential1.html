Steps to Create a Bootable USB for Windows

Materials Needed:

  • A USB flash drive (8GB minimum recommended)
  • A computer with internet access
  • Windows 10 or Windows 11 ISO file

Instructions:

  1. Download the Windows Media Creation Tool:
    • Go to the Microsoft Download page for Windows 10 or Windows 11.
    • Click on the "Download tool now" button to download the Media Creation Tool.
  2. Run the Media Creation Tool:
    • Locate the downloaded tool and double-click it to run.
    • Accept the license terms when prompted.
  3. Choose the Installation Option:
    • Select "Create installation media (USB flash drive) for another PC" and click Next.
  4. Select Language, Edition, and Architecture:
    • Choose your preferred language, edition, and select 64-bit architecture.
    • Click Next once you've made your selections.
  5. Select USB Flash Drive:
    • Choose USB flash drive as the media to use and click Next.
    • Select your USB drive from the list, ensuring itâ€™s empty.
    • Click Next to proceed.
  6. Download Windows:
    • The tool will download the Windows files to your USB drive.
  7. Create the Bootable USB:
    • Once the download is complete, the tool will create the bootable USB drive.
    • You'll see a message that says "Your USB flash drive is ready." Click Finish.
  8. Eject the USB Drive:
    • Safely eject the USB drive from your computer by right-clicking it in File Explorer and selecting Eject.

Using the Bootable USB:

  • To use the bootable USB drive, insert it into the PC where you want to install Windows.
  • The PC should boot from the USB drive, displaying the Windows installation screen.
