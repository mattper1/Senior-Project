Step 4: Install Motherboard into Case

Materials Needed

  • Motherboard
  • PC Case
  • Screws and Screwdriver

Instructions

  1. Locate the Mounting Holes in the Case

    2. Inside the case, find the mounting holes where the motherboard will be secured. These holes should align with the screw holes on the motherboard.

  2. Position the Motherboard

    3. Place the motherboard inside the case, aligning the holes on the motherboard with the mounting holes in the case. This is critical to ensure stability and proper connection with the ports on the back of the case.

  3. Check Rear IO Alignment

    4. Make sure the rear IO panel (where the USB, HDMI, and other ports are) is properly aligned with the cutout in the case. This allows access to external connections and secures the motherboard in place.

  4. Secure the Motherboard

    5. Once everything is aligned, use screws to secure the motherboard to the case at each mounting point. Tighten each screw carefully without over-tightening.
