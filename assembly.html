Step 1: CPU Installation

This close-up of the CPU shows the gold arrow. Make sure to line this up with the arrow on your motherboard during installation.

A video tutorial on how to install the CPU onto the motherboard. Handle the CPU carefully, as the pins on the underneath are fragile. Always hold the CPU from the sides, never from the bottom. When installing, avoid pressing down; instead, gently position the CPU and let it settle into place.

This image shows the CPU installation completed. Make sure everything is secure before moving on to the next steps.
